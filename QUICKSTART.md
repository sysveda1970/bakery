# クイックスタートガイド

パン屋の原価計算アプリケーションを最速で始めるためのガイドです。

## 5分で始める

### ステップ1: インストール

```bash
# プロジェクトディレクトリに移動
cd bakery-cost-calculator

# 仮想環境の作成と有効化(Windows)
python -m venv venv
venv\Scripts\activate

# または macOS/Linux
python3 -m venv venv
source venv/bin/activate

# 必要なパッケージをインストール
pip install -r requirements.txt
```

### ステップ2: アプリケーションの起動

```bash
python app.py
```

ブラウザで http://localhost:5000 にアクセス

### ステップ3: 初回セットアップ

1. **新規登録**
   - 店舗名: あなたのパン屋の名前
   - ユーザーID: ログイン用ID(3文字以上)
   - パスワード: 6文字以上

2. **材料を登録**
   - 「材料マスタ」→「材料を追加」
   - 例: 小麦粉、単価 0.5円、単位 g

3. **レシピを作成**
   - 「レシピ管理」→「新規レシピ作成」
   - 商品名、販売価格、製造個数を入力
   - 使用材料と使用量を追加

### ステップ4: 原価を確認

レシピ一覧で以下が自動計算されます:
- 材料費合計
- 1個あたり原価
- 原価率(%)
- 利益額・利益率

## 便利な機能

### 固定費の設定(オプション)

「店舗設定」から:
- 月額固定費: 300,000円(家賃、光熱費など)
- 月間生産個数: 3,000個
→ 商品1個あたり100円の固定費が自動配賦

### ラベルの作成

1. レシピ一覧から「ラベル」ボタンをクリック
2. 表示項目を選択
3. PDFをダウンロードして印刷

## よくある質問

**Q: データベースはどこに保存されますか?**
A: アプリケーションと同じディレクトリに `database.db` として保存されます。

**Q: 他のパソコンからもアクセスできますか?**
A: はい。起動したパソコンのIPアドレス(例: http://192.168.1.100:5000)でアクセスできます。

**Q: データをリセットしたい**
A: `database.db` ファイルを削除してアプリケーションを再起動してください。

**Q: パスワードを忘れました**
A: 現在はパスワードリセット機能がないため、新しいアカウントを作成してください。
